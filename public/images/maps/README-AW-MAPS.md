# AW Exo Zombies Map Images

The following map images need to be added manually for Advanced Warfare Exo Zombies:

- **outbreak-aw.webp** - Outbreak (Atlas facility)
- **infection.webp** - Infection (Burgertown)
- **carrier.webp** - Carrier (Naval carrier)
- **descent.webp** - Descent

Place WebP images in this directory. The map detail pages will display a gradient background until the images are added.
